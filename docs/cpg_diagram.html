The power of a CPG is in its four types of edges overlaying different semantic dimensions onto the same nodes — syntax structure, execution order, data flow, and control dependence. 17 edges total.

AST — Abstract syntax tree (8 edges)

Parent → child in the syntax hierarchy. The method owns its declarations and calls; each call owns its arguments.

REACHING_DEF — Data flow (3 edges)
